Who is MAPC?

Metropolitan Area Planning Council, Regional Planning Agency, serves 101 Cities and Towns in the Greater Boston Area, Transportation, Data

add logo

Why does MAPC create a bicycle and pedestrian map?

regional view instead of town-by-town, internal resource, external resource for town planner, advocacy groups and public, Who is our audience? facility map, not inventory or suitability map

Are there any related mapping initiatives?

State (MassDOT), Bicycle Network Plan (MassDOT, CTPS, Towns, Consultants)

What steps were involved?

  1. Data Collection
  2. Data Cleaning
  3. Publishing

Data Collection

101 towns, varying interests, varying data availability, schemas, quality, format

Data Cleaning

Attributes and classifications – use same categories and color schemes like other initiatives, again, be compatible

Publishing

  • Internal data availability – digital work resource for internal staff, GIS maps, analysis
  • paper map – GIS map => designer => print, updated once a year for Bike Week
  • online map – external access to internal database

What do I need to know before publishing a map online?

data collection (formats, storage), map creation, map hosting, map access (standards), commercial vs. open source tools

use what your staff knows best, staff resources are more costly than any software license, pay attention to data lock-in, allow future migrations

What tools are available to publish a map online?

commercial tools like ESRI (ArcGIS Server, ArcGIS online) and others, free like Google Maps (Google Fusion Tables) or Open Source (GeoServer, CartoDB, MapBox, Mapnik, etc.)

What is MAPC using?

Open Source, PostgreSQL/PostGIS (spatially enabled database) => TileMill for cartography => Mapnik for map rendering => Amazon S3 for map hosting => Leaflet interface for webmap and mobile map, available on GitHub
