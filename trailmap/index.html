Who is MAPC?

The Metropolitan Area Planning Council (MAPC) is a regional planning agency serving the people who live and work in the 101 cities and towns of Metropolitan Boston. Our mission is promoting smart growth principles and regional collaboration.

What steps were involved?

  1. Define Goal
  2. Data Collection
  3. Data Cleaning
  4. Map Publishing

0. Define Goal

  • Create a facility map,
    not a suitablity map or an inventory map
  • Focus on regional network,
    not data completeness or location accuracy.

1. Data Collection

Expect some data diversity in:

  • Sources
  • Interests
  • Availability
  • Schemas
  • Quality
  • Formats
101 Cities & Towns, Trustees, Park groups, OpenStreetMap etc.

2. Data Cleaning

  • Specify and migrate data into a data schema that effieciently supports your goals.
  • Stay compatible to related data collection initiatives and enable data sharing.

3. Map Publishing

  • Internal work resource (continous updates):
    Desktop GIS map
  • Public paper map (annual updates):
    Desktop GIS map → Graphic design → Print
  • Online map (continous updates):
    Data → Map rendering → Web interface

Resource Ballpark

  • Ongoing: ~10% FT
  • Print: ~40hr GIS Analyst, ~30hr Graphic Designer, ~$3,000 for 5,000 copies
  • Online: ~60hr Programmer, ~$3 per month for map tile hosting

What do I need to know to publish a map online?

  • Data storage: which spatial data storage supports my workflow?
  • Map rendering: overlay on existing basemap vs. custom map rendering?
  • Map hosting: map server for real-time rendering vs. static map images?
  • Available tools: proprietary vs. Open Source?
use what your staff knows best, staff resources are more costly than any software license, pay attention to data lock-in, allow future migrations

Lessons learned...

  • Re Open Source vs. proprietary software debate: assess staff capacity. Either leverage knowledge of existing proprietary tools and invest in software-licenses or invest in staff capacity building on Open Source tools.
  • Prevent data lock-in: some proprietary tools prefer proprietary data formats, don't go there.
  • Custom basemap design pays off in terms of long-term brand recognition.

What tools are available to publish a map online?

Some package examples, non-exhaustive.

  • ESRI: ArcGIS Online or ArcGIS Server
  • Google Maps: Fusion Tables & Maps API
  • OpenGeo: suite for web map publishing
  • CartoDB: hosted data storage solution & integrated Google Maps rendering
  • MapBox: TileMill, a cartographic design studio, for map rendering & optional integrated map hosting solution
  • ...

= Open Source Software

commercial tools like ESRI (ArcGIS Server, ArcGIS online) and others, free like Google Maps (Google Fusion Tables) or Open Source (GeoServer, CartoDB, MapBox, Mapnik, etc.)

What is MAPC using?

100% Open Source

  • PostGIS as spatial data storage
  • TileMill for cartographic design
  • Mapnik for map tile rendering
  • Amazon S3 for map tile hosting
  • Django as web framework
  • Leaflet for web map user interface

MAPC Trailmap's source code is available at: github.com/MAPC/trailmap
